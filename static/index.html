🎪 Welcome to Color Carnival🎪

Photorealistic coloring guide with palette matching, grid overlays, and pressure hints.

Walkthrough 📘

  1. Upload your pencil color chart on the Palette page.
  2. Click each swatch to pick its color, name it, and save it to a palette.
  3. Upload your reference photo on the Grid page and overlay the grid.
  4. Use the color matches to choose which pencils to use in each square.
  5. Check Pressure to see how hard to press for perfect saturation.
